messaging-app
=============
To run tests:
+ Start redis server by running
`redis-server`
+ Run app
+ Go to project root and run
`make test`

To run app,
+ Go to project root and run
`node js/server.js`

To populate db with data,
+ Start app.
+ Open console.
+ Run `populateData()` function.

To login,
+ Go to users.json
+ Pick a user
+ Enter username and password as 
`FirstNameLastName` without spaces
